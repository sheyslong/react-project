## Install

```bash
    yarn install
```
or 

```bash
    npm install
```
## Running

```bash
    yarn start
```

or 

```bash
    npm start
```

## Running Storybook 

```bash
    yarn storybook
```
or 

```bash
    npm run storybook
```
## Running Jest

```bash
    yarn test
```
or 

```bash
    npm run test
```
## Running Jest Coverage

```bash
    yarn test:coverage
```
or 

```bash
    npm run test:coverage
```
